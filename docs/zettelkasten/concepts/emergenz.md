## Definition

**Emergenz** bezeichnet das Auftreten von Eigenschaften oder Mustern auf einer höheren Systemebene, die **nicht direkt aus den Eigenschaften der einzelnen Komponenten** abgeleitet werden können.

## Emergenz in ökosemiotischen Systemen

In [[oekosemiotik|ökosemiotischen Systemen]] emergieren Muster aus der **Interaktion von Agenten mit Parameter-Oikos**.

```
Einfache Regeln + Parameter-Oikos → Komplexe emergente Muster
```

**Zentrale Erkenntnis**: Die Muster sind **nicht reduzierbar** auf initiale Bedingungen oder Regeln – sie emergieren **nur in der Interaktion mit [[parameter-as-oikos|Parameter-Oikos]]**.

## Emergente Eigenschaften in Parametrik-Labor

### Strukturelle Eigenschaften

1. **[[cluster-formation|Clusterbildung]]** - Tendenz zur Bildung räumlich konzentrierter, homogener Gruppen
2. **[[separation|Separation]]** - Tendenz zur räumlichen Trennung verschiedener Spezies/Muster
3. **Netzwerk** - Bildung verzweigter, verbundener Strukturen
4. **Kristallinität** - Geometrische, starre, hochgeordnete Muster

### Dynamische Eigenschaften

5. **[[stability|Stabilität]]** - Zeitliche Persistenz emergenter Strukturen
6. **Chaos** - Irregularität, Unvorhersagbarkeit, schnelle Veränderungen
7. **Fluidität** - Kontinuierliche Formveränderung ohne Strukturkollaps

### Quantitative Eigenschaften

8. **Dichte** - Konzentration von Trail-Intensität in Hotspots

Siehe [[../meta/map-parameter-effects.md]] für vollständige Parameter-zu-Emergenz-Matrix.

## Emergente Irreversibilität

### Kern-Mechanismus

```
Schritt 1: Agent setzt Zeichen basierend auf aktueller Umwelt
Schritt 2: Zeichen verändert Umwelt
Schritt 3: Veränderte Umwelt beeinflusst alle folgenden Agenten
Schritt 4: Diese setzen neue Zeichen basierend auf veränderter Umwelt
...
Schritt 1.500.000: Akkumulierte Ko-Konstitution ist praktisch nicht zurückverfolgbar
```

### Warum praktisch irreversibel?

1. **Nichtlineare Akkumulation**: Jede Interaktion beeinflusst zukünftige Interaktionen
2. **Parameter-Interdependenz**: Decay beeinflusst Diffusion-Effektivität
3. **Emergente Sättigung**: Selbstlimitierung durch `trailSaturation`
4. **Desynchronisation**: Individuelle Zustände verhindern Massen-Pulsieren

### Vergleich: Kryptographie vs. Emergenz

| Kryptographische Hashes | Emergente Fingerabdrücke |
|---|---|
| Mathematische Berechnung | Iterative Akkumulation |
| Theoretisch knackbar mit Rechenpower | Praktisch irreversibel durch Komplexität |
| "Rechenhärte" | "Emergente Härte" |
| Input → f(Input) = Hash | Seed → 1,5 Mio. Interaktionen → Pattern |

## Emergente Eigenschaft "Familien"

Emergente Eigenschaften korrelieren in **Familien**:

### Familie 1: Ordnung
- **Stabilität, Kristallinität, Dichte** → korreliert
- **Gefördert durch**: Hoher [[decay-rate|Decay]], hoher [[deposit-amount|Deposit]], niedriger Fade

### Familie 2: Dynamik
- **Chaos, Fluidität** → korreliert
- **Gefördert durch**: Hoher Fade, hohe [[agent-speed|Speed]], Chaos-Injection

### Familie 3: Soziale Struktur
- **Clusterbildung vs. Separation** → anti-korreliert
- **Kontrolliert durch**: Resonanz-Parameter ([[attraction-strength]], Repulsion)

Siehe [[../meta/map-parameter-effects.md#emergent-property-families]] für Details.

## Parameter-Determinierung

**These**: [[parameter-as-oikos|Parameter-Oikos]] determinieren nicht das emergente Muster, sondern **die Möglichkeitsräume der Selbstorganisation**.

**Beispiel:**
```
decayRate=0.99, diffusionFreq=8 → "Lavalampe"
decayRate=0.94, diffusionFreq=1 → "Nervensystem"
```

Gleiche Agenten, gleiche Regeln, aber **völlig verschiedene emergente Muster** durch verschiedene Parameter-Oikos.

## Kritische Schwellenwerte

Es gibt **qualitative Umschlagpunkte**, an denen emergente Eigenschaften sich fundamental ändern.

**Beispiel: Attraction/Repulsion Harmonie-Schwelle**
```
attractionStrength: 1.5 + repulsionStrength: -0.3 → Clusterbildung
attractionStrength: 1.5 + repulsionStrength: -0.7 → Segregation
```

Umschlagpunkt bei `repulsion ≈ -0.5`

Siehe [[../experiments/Experimentelle_Sektion_Index.md]] für systematische Schwellenwert-Studien.

## Chaos-Injection-Paradox

**Paradoxe Beobachtung**: Moderate Chaos-Injection kann Stabilität **erhöhen** (Loop-Breaking).

```
chaosInterval: 0 + hoher turnSpeed → Massen-Pulsieren (instabil)
chaosInterval: 300 + hoher turnSpeed → Desynchronisiert → stabiler!
```

**Theoretische Implikation**: "Störung" ist nicht immer destabilisierend – kann **Diversität erhöhen** und dadurch robustere Muster erzeugen.

## Emergenz-Metriken

Quantifizierung emergenter Eigenschaften:

- **Entropie** - Maß für Unordnung
- **Korrelationslänge** - Räumliche Ausdehnung von Strukturen
- **Fraktale Dimension** - Selbstähnlichkeit über Skalen
- **Cluster-Koeffizient** - Grad der Gruppierung
- **Pattern-Hash** - Fingerabdruck für Ähnlichkeitsvergleich

Siehe [[../methods/metrics-calculation.md]] für Implementierungsdetails.

## Schwache vs. Starke Emergenz

### Schwache Emergenz (Bedau 1997)
- Mikro-Zustand determiniert Makro-Zustand
- Aber: Makro-Muster nur durch **Simulation** zugänglich
- Keine analytische Ableitung möglich

→ **Ökosemiotische Systeme sind schwach emergent**

### Starke Emergenz
- Makro-Ebene hat kausale Kraft "nach unten"
- Umstritten in der Philosophie
